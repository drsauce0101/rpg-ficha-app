# 📜 Sistema de Fichas de RPG

Um sistema de fichas digital minimalista e funcional, projetado para permitir a edição em tempo real e persistência de dados durante sessões de RPG.

## 🛠️ Tecnologias Usadas

* **Linguagem:** Python 3.10+
* **Framework Web:** FastAPI (Backend)
* **Banco de Dados:** PostgreSQL (Persistência de dados local)
* **Interface:** HTML5, CSS3 (Estilo Dark/Medieval) e JavaScript Vanilla
* **Comunicação:** Jinja2 (Templates) e SQLModel (ORM)
* **Exposição Remota:** Ngrok (Túnel para acesso dos jogadores)
